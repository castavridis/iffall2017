Basic Selectors

Tags

Bare element tag. No #, no .. Global changes to all selected tags.

tag { css-property: value; }

Example

div {...}
<div></div>
<div></div>
<div></div>
<div></div>
<div></div>

Classes

Uses . before class name. e.g.) .selected to select <div class="selected"></div>. Use classes for styles you want to use again and again.

.class { css-property: value; }

Example

.selected {...}
.selected.blue {...}
<div></div>
<div class="selected"></div>
<div class="selected"></div>
<div class="selected blue"></div>
<div></div>

IDs

Uses # before id. e.g.) #selected-by-id. IDs are used for unique elements and can be used for other purposes.

#id { css-property: value; }

Example

.selected {...}
#selected-by-id {...}
<div></div>
<div></div>
<div id="selected-by-id" class="selected"></div>
<div class="selected"></div>
<div class="selected"></div>

Nested Selectors

Descendants

Descendants defined by a single space between each selector.

.ancestor .descendant { css-property: value; }

Example

.ancestor .descendant { ... }
<div class="ancestor"></div>

Child (Immediate Descendants)

Uses > only the children elements, or immediate descendants, matching the selector following the > are selected.

.ancestor > .descendant { css-property: value; }

Example

.ancestor > .descendant { ... }
<div class="ancestor"></div>
<div class="descendant"></div>

Adjacent Sibling

Style the very next sibling. Note that the first selector sibling is not styled but it's adjacent sibling is. Siblings must be part of the same parent element.

.sibling + .sibling { css-property: value; }

Example

.sibling-selected + .sibling-selected { ... }
<div class="sibling"></div>
<div class="sibling-selected"></div>
<div class="sibling-selected"></div>
<div class="sibling-selected"></div>
<div class="sibling"></div>

General Sibling

Style any sibling following the first selector. Siblings must be part of the same parent element.

.sibling ~ .sibling { css-property: value; }

Example

.sibling-selected ~ .sibling-selected { ... }
<div class="sibling-selected"></div>
<div class="sibling"></div>
<div class="sibling-selected"></div>
<div class="sibling"></div>
<div class="sibling-selected"></div>

Pseudo Elements

Defined by two colons ::pseudo-element. These selectors add elements the page when applied to a CSS selector.

::after

Creates a fake element that is a fake last child of it's parent element. The parent element is defined by the css-selector it's added to.

tag::after{ ... }
.class::after { ... }
#id::after { ... }

Example

.after-example::after {...}
.after-example.styled::after {...}
<div class="after-example"></div>
<div class="after-example styled"></div>

::before

Creates a psuedo-element that is the fake first child of it's parent element.

tag::before{ ... }
.class::before { ... }
#id::before { ... }

Example

.before-example::before {...}
.before-example.styled::before {...}
<div class="before-example"></div>
<div class="before-example styled"></div>
